# Getting Start

This doccument will introduce you how to use SegmentButton component to intergrate in your projects.

## Install:

```
npm install @nomentia/segment-button
```

Or

```
yarn add @nomentia/segment-button
```

## Usage:

You can use both CommonJS and ESModule to import:

```
import { SegmentButton } from '@nomentia/segment-button'
```

```
const { SegmentButton } = require('@nomentia/segment-button')
```

Check out our SegmentButton section to see how it work, and how you can implement it in your projects.
